# Checklist

- [x] 界面布局：记忆树无限画布 (PanZoom) 右下角能正常显示全局小地图 (Minimap) 并可通过点击快速归中。
- [x] 界面布局：在分析工作台点击“分屏”按钮能成功将界面分为左右两栏，并分别加载不同的历史诊断会话/企业信息。
- [x] 界面布局：按下 Cmd+K (Mac) 或 Ctrl+K (Windows) 能全局唤起命令面板，支持通过关键词搜索记忆节点或分析记录。
- [x] 功能与推演：打开“经营沙盘推演”面板，调节“碳酸锂价格”或“良品率”滑块，对应的毛利/净利图表及 AI 诊断结论能实时刷新。
- [x] 功能与推演：点击“一键导出报告”能将当前分析对话、推演结果及相关图表渲染并导出为 PDF 格式文件。
- [x] 数据接入：用户上传非标准表头的 Excel 财报后，能触发“可视化数据映射”面板并手动对齐“营业收入”、“营业成本”等系统标准字段。
- [x] 数据接入：系统配置界面中存在宏观/行业数据（如碳酸锂价格）的接入配置入口。
- [x] 可视化进阶：系统能基于 `VisualizationPayload` 渲染出针对锂电毛利拆解的“瀑布图 (Waterfall Chart)”。
- [x] 可视化进阶：在一个时间序列图表中框选/点击特定季度，其他相关图表（如成本构成饼图）会自动过滤并展示该季度数据（图表联动）。
- [x] 可视化进阶：在关键毛利暴降拐点，图表上自动显示预警线及智能批注（如“此处受原材料成本上涨 30% 影响”）。
