# 删除外部价格 API 配置模块规范

## Why
当前前端设置面板暴露了“外部数据 API”模块，要求用户手动选择宏观数据源并输入价格 API 密钥。这与系统现有“服务端私有配置收口、前端不暴露旁路诊断入口”的方向不一致，也会让用户误以为碳酸锂与六氟磷酸锂价格必须依赖手工配置才能获取。

## What Changes
- 删除前端设置面板中的“外部数据 API”模块，不再展示宏观数据源选择、碳酸锂价格 API 密钥输入、六氟磷酸锂 API 密钥输入。
- 将相关价格获取口径统一为服务端 RAG 检索链路，由系统自动获取碳酸锂、六氟磷酸锂等价格证据，前端不再提供手工接线入口。
- 价格来源说明、引用与降级状态应复用现有分析结果或证据展示区域，不新增独立配置卡片。
- **BREAKING**: 取消前端手动录入价格 API 密钥的交互口径，任何旧文案、占位提示、测试断言都不得再暗示用户需要自行填写此类密钥。

## Impact
- Affected specs: 前端设置面板规范、价格证据获取规范、RAG 来源追溯规范、最小部署口径。
- Affected code: `src/web/App.tsx` 中两处设置面板片段、可能涉及的前端测试、价格获取说明文案、服务端价格 RAG 调用与来源说明接口。

## ADDED Requirements
### Requirement: 价格信息默认由 RAG 自动获取
系统 SHALL 对碳酸锂、六氟磷酸锂等关键价格信息采用统一的服务端 RAG 获取策略，前端用户无需配置单独价格 API 密钥。

#### Scenario: 用户进入设置或配置界面
- **WHEN** 用户打开企业端或投资端的设置面板
- **THEN** 页面不展示任何“碳酸锂价格 API”“六氟磷酸锂 API”输入框，也不展示独立的“外部数据 API”配置模块

#### Scenario: 分析流程需要原材料价格证据
- **WHEN** 诊断、分析或研判流程需要引用碳酸锂或六氟磷酸锂价格
- **THEN** 服务端通过既有 RAG 链路自动检索相关价格证据，并将来源、时间或降级状态返回给前端展示

#### Scenario: RAG 暂时无法取得有效价格证据
- **WHEN** 价格 RAG 检索失败、结果离题或可信度不足
- **THEN** 系统返回明确的降级或缺失提示，且前端不得回退为要求用户手工输入 API 密钥

## MODIFIED Requirements
### Requirement: 极简配置启动
系统 SHALL 保持“前端零手工接线、服务端统一治理”的配置原则。

完整要求：前端设置界面不得暴露外部价格 API 密钥输入、宏观数据源切换等手工配置项；如需第三方凭证，应仅保留在服务端私有配置层。碳酸锂、六氟磷酸锂等价格信息默认由服务端 RAG 或已治理的数据获取链路自动提供，缺失时通过来源状态与降级提示体现，而不是要求终端用户自行补录密钥。

## REMOVED Requirements
### Requirement: 前端外部价格 API 手工配置
**Reason**: 该能力与最小部署口径、服务端私有配置治理以及 RAG 优先的价格获取策略冲突，且会增加用户理解与使用成本。
**Migration**: 删除前端模块与相关测试断言；将价格获取责任收敛到服务端 RAG 链路，并在既有结果展示区补充来源说明或降级反馈。
