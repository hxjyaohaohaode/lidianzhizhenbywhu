# Checklist

- [x] 一级记忆节点围绕中心（用户档案）呈现均匀的放射形圆环排列。
- [x] 展开一级记忆节点时，其下的二级节点在以它为中心的更大外圈（涟漪/扇形）中排列，不与其它一级节点重叠。
- [x] 用户可以通过鼠标滚轮放大和缩小整个记忆图谱的视口，图谱元素（包含文字、图标、连线）同比例缩放。
- [x] 用户可以按住背景区域（非节点）拖拽整个图谱以查看边界外的节点，拖动平滑且不会产生默认滚动条溢出。
- [x] 点击展开/收起时，节点具备“涟漪扩散”式的平滑位置过渡（Transition）。
- [x] 在缩放和平移后，点击记忆节点依然能够准确唤出覆盖全屏的 `.ndo` 详情弹窗。
