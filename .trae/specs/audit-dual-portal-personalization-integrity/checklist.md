# Checklist

- [x] 已形成覆盖前端、服务端、存储层、图表层和外部数据源的双端数据通道清单。
- [x] 每条关键数据通道都标注了企业端、普通用户端或共享底座归属，以及是否参与个性化服务。
- [x] 已识别并记录真实接口、模拟接口、降级链路和待接入占位链路，且页面展示策略一致。
- [x] 企业端首页、分析页、设置页与辅助面板已证明围绕经营目标提供个性化内容，而非仅更换角色文案。

- [x] `preferredRole`、`focusMode`、画像字段、企业基础信息与投资者基础信息已验证真实参与界面与服务差异生成。
- [x] 系统不存在企业端专属内容误展示到普通用户端，或普通用户端专属内容误展示到企业端的问题。
- [x] 图表、摘要、快捷动作、推荐内容和服务提示已验证按角色与画像差异化输出。
- [x] 已补充覆盖双端个性化、角色切换、数据来源一致性和页面差异的验证项。
