# Checklist

## 一、小数精度统一

- [ ] 所有面向用户的数值显示精确到小数点后两位，无 1 位、3 位、4 位精度的硬编码
- [ ] `roundScore` 函数使用 `toFixed(2)` 而非 `toFixed(3)`
- [ ] `data-formatter.ts` 中所有格式化函数引用全局精度常量 `GLOBAL_DECIMAL_DIGITS`
- [ ] DQI 数值显示为两位小数
- [ ] GMPS 数值显示为两位小数
- [ ] 图表中的数值标签显示为两位小数
- [ ] 表格中的数值显示为两位小数

## 二、单位选择器集成

- [ ] 首页（企业端/投资端）显示 UnitSelector 组件
- [ ] 设置页（企业端/投资端）显示完整的 UnitSelector 配置面板
- [ ] 切换金额单位后，首页所有图表和表格中的金额数值即时更新
- [ ] 切换百分比单位后，分析结果中的百分比数值即时更新
- [ ] 切换数量单位后，相关数值即时更新
- [ ] 单位偏好保存到 localStorage
- [ ] 单位偏好保存到服务端用户档案
- [ ] 页面刷新后单位偏好自动恢复

## 三、实时数据刷新

- [ ] 首页假性时间戳刷新已移除
- [ ] 首页实现真正的定时数据拉取，默认间隔 60 秒

- [ ] 首页有手动刷新按钮
- [ ] 刷新后界面数据确实更新
