# 首页图表布局重设计规格

## Why

当前首页图表布局低级、可观赏性差，企业端与投资端图表内容完全相同缺乏区分度，图表不具备动态交互能力（无法放大缩小、无弹窗详情），文字冗余掩盖了图表重点，且部分图表参数超出数学模型范围。需要全面重设计首页布局，使界面简洁美观、图表突出重点、双端差异化、交互体验专业。

## What Changes

- **首页布局全面重构**：采用"顶部核心指标卡 + 中部主图表区 + 底部辅助图表"三段式布局，减少文字、突出图表
- **企业端与投资端图表差异化**：两端首页图表内容、数据维度、展示重点完全不同，基于各自数学模型参数
- **图表类型统一为七类高级图表**：雷达图、箱型图、散点图、气泡图、热力图、桑基图、发展趋势折线图
- **所有图表参数严格限定在数学模型范围内**：企业端仅使用 GMPS 五维度（A毛利率结果/B材料成本冲击/C产销负荷/D外部风险/E现金流安全）+ DQI 三维度（盈利能力/成长能力/现金流质量）的参数；投资端使用基于 GMPS/DQI 模型推导的投资端参数
- **所有图表具备极强动态交互**：支持放大缩小、悬浮弹窗、点击详情面板
- **所有数据实时更新**：图表数据通过 API 自动刷新
- **移除冗余文字**：删除副标题、描述、脚注、强调标签等，仅保留核心标题（≤6字）

## Impact

- Affected code: `src/web/components/EnterpriseScreen.tsx`（首页布局重写）、`src/web/components/InvestorScreen.tsx`（首页布局重写）、`src/web/chart-data.ts`（图表数据配置重构）、`src/web/chart-system.tsx`（图表渲染优化）、`src/web/styles.css`（首页样式重写）、`src/web/components/ChartZoomWrapper.tsx`（缩放交互增强）
- Affected systems: 首页渲染系统、图表交互系统、数据刷新系统
- **BREAKING**: 现有 widget 配置中的 subtitle/description/footnote/emphasisTag/dataSources 字段将被忽略；企业端和投资端首页 widget 列表将完全重构

## ADDED Requirements

### Requirement: 首页三段式布局

系统 SHALL 采用"顶部核心指标卡 + 中部主图表区 + 底部辅助图表"三段式布局：

1. **顶部核心指标卡行**（高度约 120px）：横向排列 4 个核心指标卡，每个卡片仅显示指标名称、数值、状态色点，无描述文字
2. **中部主图表区**（占据主要空间）：2×2 网格布局展示 4 个主图表，每个图表占满格子，标题 ≤6 字
3. **底部辅助图表区**：1×3 网格布局展示 3 个辅助图表

布局 SHALL 满足：
- 图表区域占页面面积 85% 以上
- 卡片和图表间距统一 16px
- 所有卡片和图表使用毛玻璃效果 + 柔和阴影
- 页面无纵向滚动，一屏展示所有内容

#### Scenario: 首页一屏展示

- **WHEN** 用户进入企业端或投资端首页
- **THEN** 页面在一屏内展示所有核心指标卡和图表，无需滚动

### Requirement: 企业端首页图表配置（基于 GMPS + DQI 模型）

系统 SHALL 为企业端首页配置以下 7 个图表，所有参数严格来自 GMPS 五维度和 DQI 三维度：

**顶部核心指标卡（4个）：**
1. GMPS 毛利承压指数 — 数值 + 压力等级色点（低压绿/中压黄/高压红）
2. DQI 经营质量指数 — 数值 + 改善/稳定/恶化色点
3. 毛利率 — 当前值 + 同比变化箭头
4. 经营现金流/收入 — 当前值 + 安全垫对比色点

**中部主图表区（2×2）：**
1. **经营质量雷达图**：DQI 三维度（盈利能力 w=0.4 / 成长能力 w=0.3 / 现金流质量 w=0.3）+ GMPS 五维度（A毛利率结果 / B材料成本冲击 / C产销负荷 / D外部风险 / E现金流安全），当期值 vs 行业基准
2. **毛利承压瀑布图**：GMPS 10 个特征变量加权得分（gpmYoy=0.14, unitCostYoy=0.12, mfgCostRatio=0.12, revCostGap=0.11, saleProdRatio=0.10, liPriceYoy=0.10, invYoy=0.09, cfoRatio=0.08, lev=0.07, indVol=0.07），水平条形图按权重排序
3. **毛利率分布箱型图**：按产品线展示毛利率分布（min/Q1/median/Q3/max/outliers），各产品线毛利率数据来自企业录入
4. **经营质量热力图**：行=DQI 三维度 + GMPS 五维度，列=近4季度，色阶映射各维度得分

**底部辅助图表区（1×3）：**
5. **毛利率趋势折线图**：近4季度毛利率走势 + 行业均值参考线 + DQI=1 基准线
6. **毛利率营收散点图**：X=营收，Y=毛利率，按产品线着色，展示毛利率与营收关联
7. **成本流向桑基图**：原材料成本/人工成本/制造费用/库存费用 → 产品线 → 毛利/费用/净利润

#### Scenario: 企业端首页图表展示

- **WHEN** 企业用户进入首页
- **THEN** 页面展示 4 个核心指标卡 + 7 个图表（雷达图、瀑布图、箱型图、热力图、折线图、散点图、桑基图），所有参数来自 GMPS 和 DQI 模型

### Requirement: 投资端首页图表配置（基于 GMPS + DQI 模型推导）

系统 SHALL 为投资端首页配置以下 7 个图表，参数基于 GMPS/DQI 模型推导的投资端视角：

**顶部核心指标卡（4个）：**
1. 行业景气指数 — 数值 + 景气色点（来源：GMPS D外部风险维度 indVol）
2. 下季度承压概率 — 数值 + 风险色点（来源：GMPS 第二阶段 Logistic 输出 P_{i,t+1}）
3. 碳酸锂价格 — 当前值 + 趋势箭头（来源：GMPS B材料成本冲击维度 liPriceYoy）
4. 投资配置立场 — 保守/均衡/积极 + 信号色点（来源：DQI + GMPS 综合推导）

**中部主图表区（2×2）：**
1. **投资画像雷达图**：五维投资画像（景气弹性/盈利修复/现金流安全/估值吸引力/政策支撑），当期 vs 历史基准，维度映射 GMPS D外部风险 + DQI 盈利能力 + GMPS E现金流安全 + DQI 成长能力 + GMPS B材料成本
2. **行业风险箱型图**：按赛道展示风险得分分布（储能/动力电池/上游材料/锂电设备），风险得分来自 GMPS 各维度加权
3. **风险收益散点图**：X=风险暴露（GMPS 综合得分），Y=收益弹性（DQI 成长能力贡献），按关注标的着色
4. **风险收益热力图**：行=关注标的，列=景气/盈利/现金流/估值，色阶映射各维度得分

**底部辅助图表区（1×3）：**
5. **景气趋势折线图**：近4季度行业景气指数走势 + DQI 基准线
6. **投资三维气泡图**：X=行业景气（GMPS D维度），Y=盈利弹性（DQI 盈利能力），Z=市值规模，气泡=关注标的
7. **资金流向桑基图**：投资资金 → 赛道配置（储能/动力/材料/设备）→ 收益/风险对冲/损失

#### Scenario: 投资端首页图表展示

- **WHEN** 投资用户进入首页
- **THEN** 页面展示 4 个核心指标卡 + 7 个图表（雷达图、箱型图、散点图、热力图、折线图、气泡图、桑基图），所有参数来自 GMPS/DQI 模型推导

### Requirement: 双端图表差异化

系统 SHALL 确保企业端和投资端首页图表完全不同：

| 维度 | 企业端 | 投资端 |
|------|--------|--------|
| 核心指标 | GMPS指数/DQI指数/毛利率/现金流比率 | 景气指数/承压概率/碳酸锂价格/配置立场 |
| 雷达图 | DQI三维度+GMPS五维度经营质量 | 五维投资画像 |
| 箱型图 | 毛利率分布（按产品线） | 风险分布（按赛道） |
| 散点图 | 毛利率vs营收（按产品线） | 风险vs收益（按标的） |
| 热力图 | 经营质量（维度×季度） | 风险收益（标的×维度） |
| 折线图 | 毛利率趋势 | 景气趋势 |
| 气泡图 | 经营三维（毛利率/产销/营收） | 投资三维（景气/弹性/市值） |
| 桑基图 | 成本流向 | 资金流向 |

#### Scenario: 双端图表区分

- **WHEN** 分别查看企业端和投资端首页
- **THEN** 两端图表内容、数据维度、展示重点完全不同，无重复图表

### Requirement: 图表动态交互能力

系统 SHALL 为所有图表实现极强的动态交互：

1. **缩放功能**：
   - 鼠标滚轮缩放：30%~300% 范围
   - 缩放控件：图表右上角 +/−/重置 按钮
   - 拖拽选择区域放大，双击还原
   - 触摸屏双指缩放支持

2. **悬浮弹窗**：
   - 悬浮数据点时显示毛玻璃浮动信息卡
   - 信息卡内容：指标名称 + 当前值 + 同比/环比 + 状态标签
   - 出现/消失带缩放+淡入淡出动画（200ms）
   - 信息卡不遮挡图表主体

3. **点击详情面板**：
   - 点击图表数据元素打开沉浸式详情面板
   - 面板从点击位置缩放展开（300ms）
   - 面板内容：放大版图表 + 数据明细 + 关联指标 + 建议追问按钮
   - 支持 ESC 关闭、点击遮罩关闭
   - 面板内图表支持交互和缩放

4. **动态动画**：
   - 图表加载时数据从 0 动画增长至目标值（800ms stagger）
   - 数据更新时平滑过渡而非跳变
   - 悬停数据点放大 1.2 倍 + 发光光晕

#### Scenario: 图表缩放交互

- **WHEN** 用户在图表上滚动鼠标滚轮
- **THEN** 图表平滑缩放，坐标轴自动调整，缩放比例显示

#### Scenario: 图表悬浮弹窗

- **WHEN** 用户将鼠标悬停在图表数据点上
- **THEN** 显示毛玻璃浮动信息卡，包含指标名称、当前值、同比/环比和状态标签

#### Scenario: 图表点击详情

- **WHEN** 用户点击图表数据元素
- **THEN** 从点击位置弹出沉浸式详情面板，显示放大图表和数据明细

### Requirement: 数据实时更新

系统 SHALL 确保所有图表数据实时更新：

- 默认自动刷新间隔 60 秒
- 刷新时图表数据平滑过渡动画
- 页面右上角显示上次刷新时间
- 手动刷新按钮始终可用
- 刷新失败时显示降级提示但不影响当前展示

#### Scenario: 数据自动刷新

- **WHEN** 自动刷新间隔到达
- **THEN** 图表数据平滑过渡到最新值，刷新时间更新

### Requirement: 精简文字突出图表

系统 SHALL 大幅精简界面文字：

- **移除**：widget 的 subtitle、description、footnote、emphasisTag、dataSources 标签
- **移除**：section 副标题和强调文本
- **保留**：仅有意义的标题（≤6字）和必要的轴标签/图例
- **指标卡**：仅显示指标名称（≤4字）+ 数值 + 状态色点，无描述
- **图表容器占比**：图表区域占容器面积 85%+

#### Scenario: 界面文字精简

- **WHEN** 用户查看任意页面
- **THEN** 界面文字量减少 70%+，图表视觉占比显著提升

## MODIFIED Requirements

### Requirement: 首页布局

原首页布局从 VisualizationBoard 线性堆叠改为三段式网格布局，顶部指标卡行 + 中部2×2主图表 + 底部1×3辅助图表。

### Requirement: 图表缩放功能

原缩放范围 50%~200% 扩展为 30%~300%，增加触摸屏支持和缩放状态持久化。

### Requirement: widget 配置

widget 配置中的 subtitle、description、footnote、emphasisTag、dataSources 字段标记为废弃，渲染时不再使用。

## REMOVED Requirements

### Requirement: 冗余文字展示

**Reason**: 用户要求减少界面文字，让图表更突出
**Migration**: 移除 widget 副标题、描述、脚注、强调标签、数据源标签的渲染逻辑

### Requirement: 企业端投资端图表相同

**Reason**: 用户要求双端图表差异化，提供个性化服务
**Migration**: 企业端和投资端各自使用独立的图表配置和数据维度
