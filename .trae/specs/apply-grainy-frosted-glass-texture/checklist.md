# Checklist

- [x] 浅色模式背景呈现清透的蓝紫/青色渐变，且带有明显的颗粒噪点（Noise）质感，极度拟真参考图。
- [x] 面板和弹窗具备真实的物理磨砂玻璃效果，包含半透明高光白边、内阴影反射、高斯模糊和色彩饱和度（saturate）提升。
- [x] 所有卡片和面板的圆角、边距比例协调，具有充足的呼吸感。
- [x] 浅色模式下的投影柔和、立体，带有符合背景环境的色调。
- [x] 磨砂纹理和颗粒噪点叠加没有影响任何文字的可读性，文本对比度符合标准。
- [x] 界面细节（如图标对齐、边框粗细）在浅色模式下显得精致、高质感，无廉价感。
- [x] 切换回深色模式，深色主题依然保持其原有的深邃星空质感，未被噪点或高光边框破坏。
