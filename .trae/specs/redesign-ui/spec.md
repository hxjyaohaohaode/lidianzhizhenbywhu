# 锂电池智能诊断系统界面重新设计规格

## Why

当前系统采用深色玻璃拟态风格，视觉风格偏暗沉，缺乏现代高级感。用户需要两套独立的角色界面（企业端/投资端），每套包含四个标准页面，且首次使用时需依据数学模型参数收集数据。界面交互性不足，图标缺乏动态效果，图表不支持缩放，整体用户体验有待提升。

## What Changes

- **全面重构视觉风格**：从深色玻璃拟态切换为浅色高级现代风格，采用柔和渐变、精致阴影与微光效果
- **双角色独立界面**：企业端与投资端各自拥有独立的视觉标识与页面布局，共享组件但风格差异化
- **四页面标准架构**：首页（Dashboard）、分析页（对话工作台）、设置页、记忆中的你（Memory）补充页面
- **载入动画**：使用 `载入动画.mp4` 播放10秒后进入系统，替换当前 LoadingScreen
- **首次数据收集**：依据 GMPS 模型（毛利承压建模）和 DQI 模型（经营质量动态评价）的参数需求设计收集表单
- **图标动态交互**：所有导航图标、功能按钮实现悬停动画、状态变化反馈、操作响应效果
- **图表缩放功能**：所有 Recharts 图表组件支持整体缩放与局部细节放大
- **浅色配色方案**：主基调为浅色系，搭配品牌蓝紫渐变作为点缀色

## Impact

- Affected code: `src/web/styles.css`（全面重写）、`src/web/components/`（所有组件样式更新）、`src/web/context/AppContext.tsx`（主题变量调整）
- Affected systems: 前端渲染层、主题系统、数据收集流程、图表渲染系统
- **BREAKING**: 现有深色主题将作为可选项保留，默认切换为浅色主题

## ADDED Requirements

### Requirement: 浅色高级现代视觉风格

系统 SHALL 采用浅色配色方案作为默认主题，具备以下视觉特征：

- 主背景：柔和的浅灰蓝渐变（#F8FAFF → #EEF2FF → #F0F9FF）
- 卡片表面：白色半透明毛玻璃效果（rgba(255,255,255,0.7) + backdrop-filter: blur(20px)）
- 品牌色：蓝紫渐变（#4F6BF6 → #7C5CFC）作为强调色
- 文字层级：深色主文字（#1E293B）、次要文字（#475569）、辅助文字（#94A3B8）
- 阴影系统：多层柔和阴影（0 2px 8px rgba(79,107,246,0.06), 0 8px 24px rgba(79,107,246,0.08)）
- 圆角统一：卡片 16px、按钮 10px、输入框 10px

#### Scenario: 浅色主题渲染

- **WHEN** 用户首次进入系统
- **THEN** 界面以浅色配色方案渲染，所有组件遵循浅色设计规范

### Requirement: 双角色独立界面设计

系统 SHALL 为企业端和投资端提供视觉差异化的独立界面：

- **企业端**：以蓝绿色调为主（#4F6BF6 → #06B6D4），导航栏左侧品牌标识为 logo.png，底部角色标识为"企"
- **投资端**：以紫金色调为主（#7C5CFC → #F59E0B），导航栏左侧品牌标识为 logo.png，底部角色标识为"投"
- 两套界面共享组件代码，通过 CSS 变量和主题切换实现视觉差异化

#### Scenario: 企业端界面

- **WHEN** 用户选择企业运营分析角色
- **THEN** 系统展示蓝绿色调的企业端界面，包含企业专属的数据面板和诊断功能

#### Scenario: 投资端界面

- **WHEN** 用户选择投资人员角色
- **THEN** 系统展示紫金色调的投资端界面，包含投资专属的分析面板和推荐功能

### Requirement: 四页面标准架构

系统 SHALL 为每个角色提供四个标准页面：

1. **首页（Dashboard）**：展示关键指标卡片、趋势图表、健康度评分、诊断结论摘要
2. **分析页（对话工作台）**：AI 对话界面 + 侧边栏指标面板 + 模式切换
3. **设置页**：主题配色、数据单位、基本信息、Agent 任务配置、记忆入口
4. **记忆中的你**：知识图谱可视化 + 记忆节点管理 + 新增/编辑/删除记忆

#### Scenario: 页面导航

- **WHEN** 用户点击导航栏中的页面图标
- **THEN** 系统平滑切换到对应页面，当前页面图标显示激活状态

### Requirement: 载入动画

系统 SHALL 在启动时播放载入动画：

- 视频源：`载入动画.mp4`（位于项目根目录，需复制到 `public/loading-video.mp4`）
- 播放时长：10秒
- 播放完成后淡出过渡到角色选择页面
- 视频加载失败时显示品牌 logo + 进度条作为降级方案
- 视频播放期间显示"锂电智诊"品牌文字和底部加载提示

#### Scenario: 正常载入

- **WHEN** 系统启动
- **THEN** 播放载入动画视频，10秒后淡出并进入角色选择页面

#### Scenario: 视频加载失败

- **WHEN** 载入动画视频无法加载
- **THEN** 显示品牌 logo + 进度条，10秒后进入角色选择页面

### Requirement: 首次数据收集（基于数学模型参数）

系统 SHALL 在首次使用时收集用户数据，收集字段严格依据 GMPS 模型和 DQI 模型的参数需求：

**企业端数据收集（对应 GMPS 模型五维度 + DQI 模型三维度）：**

- A. 毛利率结果层：本季度毛利率(%)、当前季度总收入(万元)、当前季度总成本(万元)
- B. 材料成本冲击层：碳酸锂价格认知(万元/吨)
- C. 产销负荷分摊层：当前季度销量(万件)、产量(万件)、库存费用(万元)、制造费用(万元)、营业成本(万元)
- D. 外部风险传导层：（系统自动获取行业波动率）
- E. 现金流安全垫层：经营现金流(万元)、总负债(万元)、总资产(万元)
- DQI 补充：净利润(万元)、期初净资产(万元)、期末净资产(万元)、营业收入(万元)、经营活动现金流量净额(万元)
- 对比数据：去年同期毛利率(%)、去年同季度总收入(万元)、去年同季度总成本(万元)、去年同季度销量(万件)、去年同季度库存费(万元)

**投资端数据收集：**

- 投资者名称、已投资锂电企业、资金成本(年化%)、投资总金额(万元)
- 投资周期偏好、风险承受能力、关注细分领域、关注投资主题
- 其他补充说明

#### Scenario: 企业端首次数据收集

- **WHEN** 企业用户首次进入系统
- **THEN** 系统展示分步数据收集表单，字段按 GMPS 五维度和 DQI 三维度分组展示，每组标注对应的模型维度名称

#### Scenario: 投资端首次数据收集

- **WHEN** 投资用户首次进入系统
- **THEN** 系统展示投资信息收集表单，收集投资偏好和关注企业信息

### Requirement: 图标动态交互效果

系统 SHALL 为所有交互图标和按钮实现丰富的动态效果：

- **悬停动画**：图标悬停时缩放 1.15 倍 + 轻微上移 + 颜色变亮 + 阴影增强
- **状态变化反馈**：激活状态显示品牌色背景 + 左侧指示条 + 脉冲动画
- **操作响应**：点击时缩放 0.95 倍 + 涟漪效果扩散
- **导航图标**：使用 SVG 图标替代 emoji，支持颜色渐变和动画
- **按钮交互**：悬停渐变位移 + 点击缩放 + 加载状态旋转动画

#### Scenario: 图标悬停

- **WHEN** 用户将鼠标悬停在导航图标上
- **THEN** 图标执行缩放+上移+变亮动画，同时显示工具提示

### Requirement: 图表缩放功能

系统 SHALL 确保所有图表组件支持流畅的缩放功能：

- **整体缩放**：鼠标滚轮缩放图表容器，支持 50%~200% 范围
- **局部放大**：拖拽选择区域放大细节，双击还原
- **缩放控件**：图表右上角显示 +/−/重置 三个缩放按钮
- **响应式**：缩放时图表坐标轴自动调整刻度范围
- **Recharts 集成**：利用 Recharts 的 Brush 组件实现局部选择，利用自定义 wrapper 实现整体缩放

#### Scenario: 图表整体缩放

- **WHEN** 用户在图表区域滚动鼠标滚轮
- **THEN** 图表按比例缩放，坐标轴自动调整

#### Scenario: 图表局部放大

- **WHEN** 用户在图表上拖拽选择一个区域
- **THEN** 选定区域放大至整个图表区域，双击可还原

### Requirement: 角色选择页面重设计

系统 SHALL 重新设计角色选择页面：

- 顶部展示品牌 logo（logo.png）+ 系统名称"锂电智诊"
- 两个角色卡片采用浅色毛玻璃效果，悬停时卡片上浮 + 品牌色边框渐显
- 企业端卡片：蓝绿渐变图标 + "企业运营分析"标题 + 功能描述
- 投资端卡片：紫金渐变图标 + "投资人员"标题 + 功能描述
- 底部显示版本号和版权信息

#### Scenario: 角色选择

- **WHEN** 载入动画播放完毕
- **THEN** 显示角色选择页面，用户可选择企业端或投资端

## MODIFIED Requirements

### Requirement: 主题系统

原深色主题作为可选项保留，新增浅色主题作为默认值。主题切换逻辑保持不变，CSS 变量体系扩展以支持浅色方案的所有组件样式。

### Requirement: 数据收集表单

企业端数据收集表单增加模型维度标签，每个字段组标注对应的 GMPS 维度名称（A-E）和 DQI 维度名称，帮助用户理解数据用途。表单样式升级为浅色毛玻璃卡片 + 圆角输入框 + 聚焦时品牌色边框。

### Requirement: 记忆中的你页面

记忆页面背景从深色星空切换为浅色渐变 + 柔和粒子效果。节点卡片采用浅色毛玻璃样式，悬停时上浮 + 阴影增强。

## REMOVED Requirements

### Requirement: 深色主题为默认

**Reason**: 用户要求浅色配色方案为主基调
**Migration**: 深色主题保留为可选项，用户可在设置页切换
