# Checklist

- [x] 界面布局：`.viz-widget-grid` 恢复为 `display: grid` 并使用了 `grid-auto-flow: row dense`，能够像拼积木一样密集填补空白。
- [x] 界面布局：基础的指标卡（`.viz-metricCards` 等）和大型图表（`.viz-barChart`、`.viz-zebraTable` 等）有不同的宽高占比（如跨越不同的列和行），整体看起来大小错落有致（Bento Box 风格）。
- [x] 交互体验：点击某个图表右上角的“展开”按钮，该数据块会放大（如跨越多列和多行），而周围的其他图表会自动重排并填充它放大后留下的剩余空间，界面不会出现大面积空洞。
- [x] 响应式降级：当屏幕宽度缩小到一定程度（如 1100px 或更窄）时，所有卡片退化为单列流式布局，不会造成横向溢出。
