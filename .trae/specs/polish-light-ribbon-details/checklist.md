# Checklist

- [x] 光带色彩呈现平滑的金色到暖黄色渐变，无明显硬边
- [x] 光带渐变中包含微妙的淡紫/橙色调变化
- [x] 光带周围有柔和的光晕扩散，强度随距离自然衰减
- [x] 光带辉光能轻微透过节点内容显示
- [x] 光带流动曲线平滑自然，具有有机流体感
- [x] 光带流动速度有轻微随机变化，避免机械感
- [x] 不同光带间的流动相位错开，避免同步
- [x] 光带粒子有明亮的闪烁效果
- [x] 粒子大小和亮度有随机变化
- [x] 粒子运动轨迹自然，带有随机偏移
- [x] 远处光带略微模糊且透明度更高
- [x] 近处光带更清晰、色彩更饱和
- [x] 光带间有自然的遮挡关系
- [x] 光带与星空背景融合自然，无生硬边缘
- [x] 深色模式下所有优化效果正常显示
- [x] 浅色模式保持原有蓝天白云效果，不受光带优化影响
- [ ] 动画性能流畅，无明显卡顿（60fps）
