# Checklist

- [x] 双端角色视角：企业端首页与工作台优先展示经营诊断闭环，普通用户端优先展示研究决策闭环。
- [x] 数据真实性：所有关键指标、图表和结论区域都定义了来源说明、更新时间、可信度和真实/模拟标识。
- [x] 接口可用性：外部数据源异常时存在清晰的降级策略、最近成功同步时间和替代操作提示。
- [x] 按钮闭环：核心页面中的按钮、菜单、快捷动作和导出入口均有明确的业务结果与反馈状态。
- [x] 界面美观：双端页面具备统一的视觉规范、卡片层级、留白节奏和高质感动态效果约束。
- [x] 布局稳定：主要页面在常见桌面分辨率下不会出现图表溢出、遮挡、错位或悬浮层层级异常。
- [x] 图表规范：图表具备统一色板、单位、空态、异常提示、来源信息、联动反馈和导出规则。
- [x] 功能可行：新增竞争力模块均标注了数据依赖、首版范围和实现前提，不包含无法落地的设想型功能。
- [x] 体验验收：规格中包含发布前的双端体验门禁，能用于产品、设计、研发共同验收。
