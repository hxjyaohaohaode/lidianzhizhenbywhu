# 螺旋记忆树与多级展开交互 Spec

## Why
当前系统中的“记忆中的你”界面（`MemoryScreen`）节点采用简单的静态散点布局或随机布局，缺乏清晰的层级结构与分类。随着系统记录的用户画像和长期记忆越来越多，用户难以直观地理解不同记忆之间的关联。通过将节点重构为以用户档案为中心的“螺旋形（Spiral）多级树状展开”结构，不仅能提升视觉美感，还能建立清晰的“中心（用户） -> 一级（大类） -> 二级（细节）”浏览心智。

## What Changes
- 重构 `MemoryScreen` 的节点生成与渲染逻辑，采用基于黄金螺旋或阿基米德螺旋的坐标计算方式。
- 将节点结构改造为多层级（Multi-level）。根节点（中心）为“用户档案”，其下挂载多个一级节点（如：投资画像、关注领域、交互偏好等），一级节点下挂载具体的二级记忆节点（如：具体的偏好参数、最近的分析结论）。
- 实现节点的展开/折叠交互：默认只显示中心节点与一级节点；点击一级节点后，动态呈螺旋或放射状展开其所属的二级节点。
- 更新数据流，使记忆节点直接对接后端系统返回的真实 `UserProfile` 与长期记忆（Memory Mementos），实现实时更新。
- 每个节点卡片均显示记忆内容的摘要，点击最叶子节点（或任意带有详情的节点）时，仍通过现有的 `.ndo` 弹窗展示详细记忆内容。
- 补充与多级节点连线（SVG lines）相关的坐标计算逻辑，保证父子节点间的连线在展开/折叠时动态更新。

## Impact
- Affected specs: 记忆页布局与动效、用户画像与长期记忆的数据绑定
- Affected code: `src/web/App.tsx`, `src/web/styles.css`

## ADDED Requirements
### Requirement: 多级螺旋展开交互
系统 SHALL 在记忆页以中心向外辐射的螺旋路径渲染记忆节点。支持层级折叠与展开。

#### Scenario: 点击展开二级记忆
- **WHEN** 用户在记忆页点击了一个一级记忆节点（例如“投资偏好”）
- **THEN** 该节点周围沿螺旋路径动态呈现属于它的二级子节点（如“偏好中等风险”、“关注现金流”），同时 SVG 连线正确连接父子节点。

### Requirement: 实时数据映射与摘要显示
系统 SHALL 从实时的 `userProfile` 和后端会话状态中提取记忆数据，而非使用硬编码。每个生成的节点必须在卡片表面展示该记忆的核心摘要。

## MODIFIED Requirements
### Requirement: 记忆详情查看
[保持现有能力，但适配新层级] 无论是中心节点、一级节点还是展开的二级节点，只要包含详尽的描述内容（`desc`），点击（或在双击/详情按钮点击时，视交互冲突而定） SHALL 唤起居中的记忆详情弹窗显示完整内容。
