# Checklist

- [x] 进入“记忆页”时，首先渲染中心用户档案节点和第一级分类节点，并呈现明确的层级与连线关系。
- [x] 节点的排列遵循螺旋或向外扩散的美观布局算法，避免节点在视觉上大量重叠。
- [x] 点击一级分类节点时，能动态展开其下的二级记忆内容，且连线和坐标随之刷新，布局依然合理。
- [x] 每个节点在界面上能直接阅读到内容摘要（Summary）。
- [x] 在需要查看详细内容的节点上，能够唤起 `.ndo` 的模态弹层查看其详细描述（Detail / Desc）。
- [x] 记忆内容通过传入的 `userProfile` 或后端上下文获取，而不是固定的假数据（Mock Data）。
