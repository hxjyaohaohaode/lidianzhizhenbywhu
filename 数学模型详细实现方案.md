## 一、经营质量变化的动态评价模型（DQI模型）

### 1.1 模型概述

**模型目标**：构建动态经营质量指数（Dynamic Quality Index, DQI），刻画企业经营质量的演进方向与变化幅度。

**核心理念**：经营质量不仅取决于企业当前盈利能力与成长水平，更体现在资源配置效率及其随时间的变化趋势。

### 1.2 指标体系构建

从三个维度选取核心指标：

| 维度    | 指标      | 变量名    | 数据来源        |
| ----- | ------- | ------ | ----------- |
| 盈利能力  | 净资产收益率  | ROE    | 利润表 + 资产负债表 |
| 成长能力  | 营收增长率   | Growth | 利润表         |
| 现金流质量 | 经营现金流净额 | OCF    | 现金流量表       |

### 1.3 核心公式实现

#### 1.3.1 ROE 计算

```python
def calculate_roe(net_profit, beginning_equity, ending_equity):
    """
    净资产收益率计算
    
    公式：ROE = 净利润 / 平均净资产 × 100%
    其中：平均净资产 = (期初净资产 + 期末净资产) / 2
    
    参数:
        net_profit: 净利润
        beginning_equity: 期初净资产
        ending_equity: 期末净资产
    
    返回:
        ROE: 净资产收益率（百分比）
    """
    average_equity = (beginning_equity + ending_equity) / 2
    roe = (net_profit / average_equity) * 100
    return roe
```

#### 1.3.2 经营效率计算

```python
def calculate_operating_efficiency(roe, ocf, alpha=0.5, beta=0.5):
    """
    经营效率计算
    
    公式：E_t = α · ROE_t + β · OCF_t
    
    参数:
        roe: 净资产收益率（百分比）
        ocf: 经营现金流净额（标准化后）
        alpha: 盈利能力权重（默认0.5）
        beta: 现金流权重（默认0.5）
    
    返回:
        E: 经营效率指数
    """
    # 注意：OCF需要标准化处理，建议除以营业收入
    E = alpha * roe + beta * ocf
    return E
```

#### 1.3.3 动态经营质量指数（DQI）

```python
def calculate_dqi(current_period_data, previous_period_data):
    """
    动态经营质量指数计算
    
    公式：DQI_t = E_t / E_{t-1}
    
    或分解形式：
    DQI_t = w1 · (ROE_t / ROE_{t-1}) + w2 · (Growth_t / Growth_{t-1}) + w3 · (OCF_t / OCF_{t-1})
    
    权重设置：
    - w1 = 0.4（盈利能力）
    - w2 = 0.3（成长能力）
    - w3 = 0.3（现金流质量）
    
    参数:
        current_period_data: 当期数据字典
        previous_period_data: 上期数据字典
    
    返回:
        dqi: 动态经营质量指数
        decomposition: 各维度贡献分解
    """
    # 方法1：基于经营效率比值
    E_current = calculate_operating_efficiency(
        current_period_data['roe'],
        current_period_data['ocf_ratio']
    )
    E_previous = calculate_operating_efficiency(
        previous_period_data['roe'],
        previous_period_data['ocf_ratio']
    )
    dqi_method1 = E_current / E_previous if E_previous != 0 else None
    
    # 方法2：分解计算（推荐）
    # 权重
    w1, w2, w3 = 0.4, 0.3, 0.3
    
    # 计算各维度变化率
    roe_ratio = current_period_data['roe'] / previous_period_data['roe'] if previous_period_data['roe'] != 0 else 1
    growth_ratio = current_period_data['growth'] / previous_period_data['growth'] if previous_period_data['growth'] != 0 else 1
    ocf_ratio = current_period_data['ocf_ratio'] / previous_period_data['ocf_ratio'] if previous_period_data['ocf_ratio'] != 0 else 1
    
    # 加权求和
    dqi = w1 * roe_ratio + w2 * growth_ratio + w3 * ocf_ratio
    
    # 分解贡献
    decomposition = {
        'profitability_contribution': w1 * roe_ratio,
        'growth_contribution': w2 * growth_ratio,
        'cashflow_contribution': w3 * ocf_ratio,
        'roe_ratio': roe_ratio,
        'growth_ratio': growth_ratio,
        'ocf_ratio': ocf_ratio
    }
    
    return {
        'dqi': dqi,
        'decomposition': decomposition
    }
```

#### 1.3.4 驱动因素识别

```python
def identify_driving_factor(roe_ratio, growth_ratio, ocf_ratio):
    """
    驱动因素识别
    
    公式：Driver = argmax{ROE_t/ROE_{t-1}, Growth_t/Growth_{t-1}, OCF_t/OCF_{t-1}}
    
    判别规则：
    - 若 ROE比率最大 → 盈利能力驱动
    - 若 Growth比率最大 → 成长能力驱动
    - 若 OCF比率最大 → 现金流驱动
    
    参数:
        roe_ratio: ROE变化率
        growth_ratio: 营收增长率变化率
        ocf_ratio: OCF变化率
    
    返回:
        driver: 驱动因素名称
        ratios: 各维度比率
    """
    ratios = {
        '盈利能力': roe_ratio,
        '成长能力': growth_ratio,
        '现金流质量': ocf_ratio
    }
    
    driver = max(ratios, key=ratios.get)
    
    return {
        'driver': driver,
        'ratios': ratios
    }
```

#### 1.3.5 变化率阈值判别

```python
def classify_change_magnitude(ratio):
    """
    变化率阈值判别
    
    判别规则：
    - r_i < 1 ⇒ 该指标为负向影响
    - 下降幅度 d_i = 1 - r_i
      - d_i > 0.2 → 显著下降（强风险）
      - 0.1 < d_i ≤ 0.2 → 中度下降
      - d_i ≤ 0.1 → 轻微波动
    
    参数:
        ratio: 变化率
    
    返回:
        classification: 分类结果
        decline_rate: 下降幅度
    """
    if ratio >= 1:
        return {
            'classification': '正向影响',
            'decline_rate': 0,
            'risk_level': 'low'
        }
    else:
        decline_rate = 1 - ratio
        
        if decline_rate > 0.2:
            risk_level = 'high'
            classification = '显著下降（强风险）'
        elif decline_rate > 0.1:
            risk_level = 'medium'
            classification = '中度下降'
        else:
            risk_level = 'low'
            classification = '轻微波动'
        
        return {
            'classification': classification,
            'decline_rate': decline_rate,
            'risk_level': risk_level
        }
```

```python
def interpret_dqi(dqi):
    """
    DQI 判别标准

            'status': '稳定',
            'description': '经营质量基本稳定',
            'trend': 'stable'
        }
    else:
        return {
            'status': '恶化',

            'trend': 'negative'
        }
```

#### 1.3.7 多期趋势分析

```python
def analyze_trend(dqi_series):
    """
    多期趋势分析
    
    输入：连续年份的DQI序列 {DQI_{t-2}, DQI_{t-1}, DQI_t}
    
    判别规则：
    - 连续大于1：经营质量持续改善
    - 连续小于1：经营质量持续恶化
    - 波动变化：经营状态不稳定
    
    参数:
        dqi_series: DQI时间序列列表
    
    返回:
        trend_analysis: 趋势分析结果
    """
    if len(dqi_series) < 2:
        return {'trend': '数据不足', 'stability': 'unknown'}
    
    # 判断每个时期的状态
    states = ['improving' if dqi > 1 else 'deteriorating' if dqi < 1 else 'stable' 
              for dqi in dqi_series]
    
    # 判断连续性
    all_improving = all(s == 'improving' for s in states)
    all_deteriorating = all(s == 'deteriorating' for s in states)
    
    if all_improving:
        trend = '持续改善'
        stability = 'high'
    elif all_deteriorating:
        trend = '持续恶化'
        stability = 'high'
    else:
        trend = '波动变化'
        stability = 'low'
    
    return {
        'trend': trend,
        'stability': stability,
        'states': states,
        'latest_dqi': dqi_series[-1]
    }
```

## 二、毛利承压评估预测模型（GMPS模型）

### 2.1 模型概述

**模型目标**：构建兼具行业机理解释性和前瞻预警能力的毛利承压诊断模型。

**两阶段设计**：

- **第一阶段**：评价企业当前季度的毛利承压程度与主要压力来源
- **第二阶段**：预测企业下一季度发生持续性毛利承压的概率

### 2.2 特征因素体系

#### 2.2.1 A层：毛利率结果

```python
def calculate_gpm_yoy(current_gpm, previous_gpm):
    """
    毛利率同比变化
    
    公式：GPM_YoY = g(GPM_t) = (GPM_t - GPM_{t-4}) / GPM_{t-4}
    
    参数:
        current_gpm: 当期毛利率
        previous_gpm: 去年同期毛利率（t-4期）
    
    返回:
        gpm_yoy: 毛利率同比变化率
    """
    if previous_gpm == 0:
        return None
    gpm_yoy = (current_gpm - previous_gpm) / previous_gpm
    return gpm_yoy

def calculate_rev_cost_gap(current_revenue, previous_revenue, 
                           current_cost, previous_cost):
    """
    营收增速减成本增速
    
    公式：rev_cost_gap = g(Revenue_{i,t}) - g(Cost_{i,t})
    其中：g(X_{i,t}) = (X_{i,t} - X_{i,t-4}) / X_{i,t-4}
    
    解释：若为负，表示成本扩张快于收入，通常意味着毛利承压
    
    参数:
        current_revenue: 当期营业收入
        previous_revenue: 去年同期营业收入
        current_cost: 当期营业成本
        previous_cost: 去年同期营业成本
    
    返回:
        rev_cost_gap: 营收增速与成本增速的差额
    """
    revenue_growth = (current_revenue - previous_revenue) / previous_revenue if previous_revenue != 0 else 0
    cost_growth = (current_cost - previous_cost) / previous_cost if previous_cost != 0 else 0
    
    rev_cost_gap = revenue_growth - cost_growth
    return rev_cost_gap
```

#### 2.2.2 B层：材料成本冲击

```python
def calculate_li_price_yoy(current_li_price, previous_li_price):
    """
    碳酸锂价格同比变化
    
    公式：g(LiPrice_t)
    
    解释：上游供应商的锂盐价格上升通常会压缩中下游企业毛利
    
    参数:
        current_li_price: 当期碳酸锂价格
        previous_li_price: 去年同期碳酸锂价格
    
    返回:
        li_price_yoy: 碳酸锂价格同比变化率
    """
    if previous_li_price == 0:
        return None
    li_price_yoy = (current_li_price - previous_li_price) / previous_li_price
    return li_price_yoy

def calculate_unit_cost_yoy(current_cost, current_sales, 
                            previous_cost, previous_sales):
    """
    单位营业成本变化率（同比）
    
    公式：[(Cost_{i,t}/Sale_{i,t}) - (Cost_{i,t-4}/Sale_{i,t-4})] / (Cost_{i,t-4}/Sale_{i,t-4})
    
    解释：反映单位产品成本是否明显抬升，把"外部材料涨价"和"企业内部传导结果"统一起来
    
    参数:
        current_cost: 当期营业成本
        current_sales: 当期销量
        previous_cost: 去年同期营业成本
        previous_sales: 去年同期销量
    
    返回:
        unit_cost_yoy: 单位成本同比变化率
    """
    current_unit_cost = current_cost / current_sales if current_sales != 0 else 0
    previous_unit_cost = previous_cost / previous_sales if previous_sales != 0 else 0
    
    if previous_unit_cost == 0:
        return None
    
    unit_cost_yoy = (current_unit_cost - previous_unit_cost) / previous_unit_cost
    return unit_cost_yoy
```

#### 2.2.3 C层：产销负荷与制造费用分摊

```python
def calculate_inv_yoy(current_inventory, previous_inventory):
    """
    库存同比增速
    
    公式：g(Inventory_{i,t}) = (Inventory_{i,t} - Inventory_{i,t-4}) / Inventory_{i,t-4}
    
    参数:
        current_inventory: 当期库存
        previous_inventory: 去年同期库存
    
    返回:
        inv_yoy: 库存同比增速
    """
    if previous_inventory == 0:
        return None
    inv_yoy = (current_inventory - previous_inventory) / previous_inventory
    return inv_yoy

def calculate_sale_prod_ratio(sales, production):
    """
    产销率
    
    公式：Sale_{i,t} / Production_{i,t}
    
    参数:
        sales: 销量
        production: 产量
    
    返回:
        sale_prod_ratio: 产销率
    """
    if production == 0:
        return None
    sale_prod_ratio = sales / production
    return sale_prod_ratio

def calculate_mfg_cost_ratio(manufacturing_expense, operating_cost):
    """
    制造费用占营业成本比
    
    公式：MfgExp_{i,t} / Cost_{i,t}
    
    参数:
        manufacturing_expense: 制造费用
        operating_cost: 营业成本
    
    返回:
        mfg_cost_ratio: 制造费用占比
    """
    if operating_cost == 0:
        return None
    mfg_cost_ratio = manufacturing_expense / operating_cost
    return mfg_cost_ratio
```

#### 2.2.4 D层：外部风险传导

```python
def calculate_ind_vol(index_data, window=20):
    """
    锂电行业指数波动率
    
    公式：IndVol_t
    
    解释：可用沪深300波动率、创业板指数波动率，反映资本市场整体风险偏好变化
    
    参数:
        index_data: 指数时间序列数据
        window: 计算窗口期（默认20个交易日）
    
    返回:
        ind_vol: 行业指数波动率
    """
    import numpy as np
    
    # 计算收益率
    returns = np.diff(np.log(index_data))
    
    # 计算波动率（标准差）
    ind_vol = np.std(returns[-window:]) * np.sqrt(252)  # 年化波动率
    
    return ind_vol
```

```python
def calculate_cfo_ratio(operating_cash_flow, revenue):
    """
    经营现金流/营业收入
    
    公式：CFO_{i,t} / Revenue_{i,t}
    
    解释：该值越低，表示收入转化为现金的能力越弱
    
    参数:
        operating_cash_flow: 经营活动现金流量净额
        revenue: 营业收入
    
    返回:
        cfo_ratio: 现金流比率
    """
    if revenue == 0:
        return None
    cfo_ratio = operating_cash_flow / revenue
    return cfo_ratio

def calculate_leverage(total_liabilities, total_assets):
    """
    资产负债率
    
    公式：Liabilities_{i,t} / Assets_{i,t}
    
    解释：值越高，财务缓冲越弱
    
    参数:
        total_liabilities: 总负债
        total_assets: 总资产
    
    返回:
        leverage: 资产负债率
    """
    if total_assets == 0:
        return None
    leverage = total_liabilities / total_assets
    return leverage
```

### 2.3 第一阶段：GMPS计算

#### 2.3.1 标准化打分函数

```python
def score_increasing_risk(x, L, H):
    """
    越大越危险的变量打分函数
    
    公式：
    score(x) = 20,                           x ≤ L
    score(x) = 20 + 60·(x-L)/(H-L),          L < x < H
    score(x) = 80,                           x ≥ H
    
    解释：
    - 小于低风险阈值L：低风险（20分）
    - 大于高风险阈值H：高风险（80分）
    
    参数:
        x: 变量值
        L: 低风险阈值
        H: 高风险阈值
    
    返回:
        score: 风险分数（0-100）
    """
    if x <= L:
        return 20
    elif x >= H:
        return 80
    else:
        return 20 + 60 * (x - L) / (H - L)

def score_decreasing_risk(x, L, H):
    """
    越小越危险的变量打分函数
    
    公式：
    score(x) = 80,                           x ≤ L
    score(x) = 20 + 60·(H-x)/(H-L),          L < x < H
    score(x) = 20,                           x ≥ H
    
    参数:
        x: 变量值
        L: 低风险阈值
        H: 高风险阈值
    
    返回:
        score: 风险分数（0-100）
    """
    if x <= L:
        return 80
    elif x >= H:
        return 20
    else:
        return 20 + 60 * (H - x) / (H - L)
```

#### 2.3.3 GMPS 计算主函数

```python
def calculate_gmps(features):
    """
    毛利承压指数（GMPS）计算
    
    公式：
    GMPS_{i,t} = Σ_{k=1}^{10} w_k · score_{i,t}^{(k)}
    
    展开：
    GMPS = 0.14·score(gpm_yoy) + 0.11·score(rev_cost_gap)
         + 0.10·score(li_price_yoy) + 0.12·score(unit_cost_yoy)
         + 0.09·score(inv_yoy) + 0.10·score(sale_prod_ratio)
         + 0.12·score(mfg_cost_ratio) + 0.07·score(ind_vol)
         + 0.08·score(cfo_ratio) + 0.07·score(lev)
    
    参数:
        features: 包含10个特征值的字典
    
    返回:
        gmps_result: GMPS计算结果及分解
    """
    scores = {}
    weighted_scores = {}
    
    for feature_name, feature_value in features.items():
        if feature_name not in FEATURE_THRESHOLDS:
            continue
        
        config = FEATURE_THRESHOLDS[feature_name]
        
        # 根据类型选择打分函数
        if config['type'] == 'increasing_risk':
            score = score_increasing_risk(feature_value, config['L'], config['H'])
        else:
            score = score_decreasing_risk(feature_value, config['L'], config['H'])
        
        scores[feature_name] = score
        weighted_scores[feature_name] = score * config['weight']
    
    # 计算GMPS总分
    gmps = sum(weighted_scores.values())
    
    # 计算各维度子得分
    dimension_scores = calculate_dimension_scores(scores)
    
    # 等级划分
    level = classify_gmps_level(gmps)
    
    return {
        'gmps': gmps,
        'level': level,
        'scores': scores,
        'weighted_scores': weighted_scores,
        'dimension_scores': dimension_scores
    }

def calculate_dimension_scores(scores):
    """
    计算各维度子得分
    
    公式：S_K = Σ_{j=1}^{n} ω_{k,j} × score_j / Σ_{j=1}^{n} ω_{k,j}
    
    五个维度：
    - A层：毛利率结果
    - B层：材料成本冲击
    - C层：产销负荷与制造费用分摊
    - D层：外部风险传导
    - E层：现金流与安全垫层
    """
    dimensions = {
        'A_毛利率结果': ['gpm_yoy', 'rev_cost_gap'],
        'B_材料成本冲击': ['li_price_yoy', 'unit_cost_yoy'],
        'C_产销负荷': ['inv_yoy', 'sale_prod_ratio', 'mfg_cost_ratio'],
        'D_外部风险': ['ind_vol'],
        'E_现金流安全': ['cfo_ratio', 'lev']
    }
    
    dimension_scores = {}
    
    for dim_name, feature_list in dimensions.items():
        total_score = 0
        total_weight = 0
        
        for feature in feature_list:
            if feature in scores and feature in FEATURE_THRESHOLDS:
                weight = FEATURE_THRESHOLDS[feature]['weight']
                total_score += scores[feature] * weight
                total_weight += weight
        
        if total_weight > 0:
            dimension_scores[dim_name] = total_score / total_weight
        else:
            dimension_scores[dim_name] = 0
    
    return dimension_scores

def classify_gmps_level(gmps):
    """
    GMPS等级划分
    
    判别规则：
    - GMPS < 40 → 低压
    - 40 ≤ GMPS < 70 → 中压
    - GMPS ≥ 70 → 高压
    
    参数:
        gmps: 毛利承压指数
    
    返回:
        level: 承压等级
    """
    if gmps < 40:
        return '低压'
    elif gmps < 70:
        return '中压'
    else:
        return '高压'
```

#### 2.4.1 Logistic回归模型

```python
def predict_pressure_probability(gmps, dimension_scores, beta_params):
    """
    Logistic回归预测下一季度承压概率
    
    公式：
    P(Y_{i,t+1} = 1 | X_{i,t}) = σ(β_0 + β^T X_{i,t})
    
    其中：
    σ(x) = 1 / (1 + e^{-x})
    
    展开：
    P_{i,t+1} = 1 / (1 + exp[-(β_0 + β_1·GMPS_{i,t} + Σ_{d=2}^{6} β_d·S^{(d)})])
    
    参数:
        gmps: 当前GMPS总分
        dimension_scores: 五维得分字典
        beta_params: 回归参数（β_0, β_1, β_2, ..., β_6）
    
    返回:
        probability: 下一季度承压概率
    """
    import math
    
    # 提取参数
    beta_0 = beta_params['beta_0']
    beta_1 = beta_params['beta_1']
    beta_2 = beta_params.get('beta_2', 0)  # A层
    beta_3 = beta_params.get('beta_3', 0)  # B层
    beta_4 = beta_params.get('beta_4', 0)  # C层
    beta_5 = beta_params.get('beta_5', 0)  # D层
    beta_6 = beta_params.get('beta_6', 0)  # E层
    
    # 计算线性组合
    linear_combination = (
        beta_0 + 
        beta_1 * gmps + 
        beta_2 * dimension_scores.get('A_毛利率结果', 0) +
        beta_3 * dimension_scores.get('B_材料成本冲击', 0) +
        beta_4 * dimension_scores.get('C_产销负荷', 0) +
        beta_5 * dimension_scores.get('D_外部风险', 0) +
        beta_6 * dimension_scores.get('E_现金流安全', 0)
    )
    
    # Sigmoid函数
    probability = 1 / (1 + math.exp(-linear_combination))
    
    return probability

def classify_risk_level(probability):
    """
    风险等级分类
    
    判别规则：
    - P < 0.33 → 低风险
    - 0.33 ≤ P < 0.66 → 中风险
    - P ≥ 0.66 → 高风险
    
    参数:
        probability: 承压概率
    
    返回:
        risk_level: 风险等级
    """
    if probability < 0.33:
        return '低风险'
    elif probability < 0.66:
        return '中风险'
    else:
        return '高风险'
```

```python
class DiagnosticModelEngine:
    """
    诊断模型引擎
    整合DQI模型和GMPS模型
    """
    
    def __init__(self):
        self.dqi_model = DQIModel()
        self.gmps_model = GMPSModel()
    
    def run_full_diagnosis(self, enterprise_data, industry_data):
        """
        执行完整诊断
        
        参数:
            enterprise_data: 企业财务数据
            industry_data: 行业外部数据
        
        返回:
            diagnosis_result: 完整诊断结果
        """
        # 1. 计算DQI（经营质量）
        dqi_result = self.dqi_model.calculate(enterprise_data)
        
        # 2. 计算GMPS（毛利承压）
        gmps_result = self.gmps_model.calculate(enterprise_data, industry_data)
        
        # 3. 预测下一季度风险
        probability = self.gmps_model.predict_probability(
            gmps_result['gmps'],
            gmps_result['dimension_scores']
        )
        
        # 4. 综合诊断结论
        diagnosis_result = {
            'operating_quality': {
                'dqi': dqi_result['dqi'],
                'status': dqi_result['status'],
                'driver': dqi_result['driver'],
                'trend': dqi_result['trend']
            },
            'gross_margin_pressure': {
                'gmps': gmps_result['gmps'],
                'level': gmps_result['level'],
                'dimension_scores': gmps_result['dimension_scores'],
                'probability_next_quarter': probability,
                'risk_level': classify_risk_level(probability)
            },
            'recommendations': self.generate_recommendations(dqi_result, gmps_result)
        }
        
        return diagnosis_result
    
    def generate_recommendations(self, dqi_result, gmps_result):
        """
        生成改进建议
        
        基于两个模型的结果，生成针对性的改进建议
        """
        recommendations = []
        
        # 基于DQI的建议
        if dqi_result['dqi'] < 1:
            driver = dqi_result['driver']
            if driver == '盈利能力':
                recommendations.append('建议优化成本结构，提高盈利能力')
            elif driver == '成长能力':
                recommendations.append('建议拓展市场，提升营收增长')
            else:
                recommendations.append('建议加强现金流管理')
        
        # 基于GMPS的建议
        if gmps_result['level'] in ['中压', '高压']:
            dim_scores = gmps_result['dimension_scores']
            max_dim = max(dim_scores, key=dim_scores.get)
            
            if '材料成本' in max_dim:
                recommendations.append('建议优化供应链，降低原材料成本')
            elif '产销负荷' in max_dim:
                recommendations.append('建议优化库存管理，提高产销匹配度')
            elif '现金流' in max_dim:
                recommendations.append('建议加强现金流管理，提高财务安全性')
        
        return recommendations
```

### 3.2 模型验证机制

```python
def validate_model_inputs(enterprise_data, required_fields):
    """
    模型输入验证
    
    确保所有必需数据完整且合理
    """
    missing_fields = []
    invalid_fields = []
    
    for field in required_fields:
        if field not in enterprise_data:
            missing_fields.append(field)
        elif enterprise_data[field] is None:
            missing_fields.append(field)
        elif not isinstance(enterprise_data[field], (int, float)):
            invalid_fields.append(field)
    
    return {
        'valid': len(missing_fields) == 0 and len(invalid_fields) == 0,
        'missing_fields': missing_fields,
        'invalid_fields': invalid_fields
    }

def validate_calculation_result(result, model_type):
    """
    模型计算结果验证
    
    确保计算结果在合理范围内
    """
    if model_type == 'DQI':
        # DQI应该在合理范围内（通常0.5-2.0）
        if 'dqi' in result and (result['dqi'] < 0 or result['dqi'] > 5):
            return {'valid': False, 'reason': 'DQI值超出合理范围'}
    
    elif model_type == 'GMPS':
        # GMPS应该在0-100之间
        if 'gmps' in result and (result['gmps'] < 0 or result['gmps'] > 100):
            return {'valid': False, 'reason': 'GMPS值超出合理范围'}
    
    return {'valid': True}
```

```python
class ModelParameterManager:
    """
    模型参数管理器
    支持参数的动态调整和持久化
    """
    
    def __init__(self, config_file='model_config.yaml'):
        self.config = self.load_config(config_file)
    
    def update_weights(self, model_type, new_weights):
        """
        更新模型权重
        
        参数:
            model_type: 'dqi' 或 'gmps'
            new_weights: 新的权重字典
        """
        if model_type == 'dqi':
            self.config['dqi_model']['weights'].update(new_weights)
            # 验证权重和为1
            total = sum(self.config['dqi_model']['weights'].values())
            if abs(total - 1.0) > 0.01:
                raise ValueError(f"权重总和必须为1，当前为{total}")
        
        elif model_type == 'gmps':
            self.config['gmps_model']['feature_weights'].update(new_weights)
            # 验证权重和为1
            total = sum(self.config['gmps_model']['feature_weights'].values())
            if abs(total - 1.0) > 0.01:
                raise ValueError(f"权重总和必须为1，当前为{total}")
    
    def save_config(self):
        """
        保存配置到文件
        """
        import yaml
        with open('model_config.yaml', 'w', encoding='utf-8') as f:
            yaml.dump(self.config, f, allow_unicode=True)
```
